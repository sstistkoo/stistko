ℹ️ Jak fungují jazykové značky

Všechny jazykové zkratky ve vstupním souboru budou automaticky nahrazeny cílovým jazykem:

  • Vstup (CZ): "KJV překlady (CZ)" → Výstup (SK): "KJV preklady (SK)"
  • Vstup CZ: "Verze CZ" → Výstup SK: "Verzia SK"
  • Funguje pro: (CZ), CZ, (EN), EN, (SK), SK, (DE), DE, atd.
  • Slova jazyků: "Czech / English / Slovak / Polish" se přepíší na cílový jazyk.

Vyberte cílové jazyky:

Doporučený vstup je cs.json. CZ zkratky i slovo "Czech" zůstávají ošetřené a mapují se na cílový jazyk.
