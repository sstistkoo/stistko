O čem aplikace je
Tento nástroj překládá hesla Strong slovníku (řečtina/hebrejština) do češtiny po dávkách přes AI provider (`Groq`, `Gemini`, `OpenRouter`). Umí ukládat průběh, zastavit AUTO podle tokenů, importovat/exportovat TXT/JSON a znovu zpracovat neúspěšné položky.
Jak získat API klíče
`Groq`: vytvoř klíč v console.groq.com/keys.
`Gemini`: vytvoř klíč v AI Studio.
`OpenRouter`: vytvoř klíč v openrouter.ai/keys.
Pak vlož klíč do pole API a můžeš ho uložit do knihovny tlačítkem Uložit klíč.
Knihovna API klíčů
Pro každého providera můžeš mít více klíčů. Vyber klíč v rozbalovacím menu pod API polem a přepni ho jedním klikem. Aktivní klíč se používá pro překlad i kontrolu limitů.
Limity a kvóty
Tlačítko `📊 Limity` zobrazuje dostupné údaje o kreditech/limitech z API providera a doplňkové odhady. U některých providerů nejsou všechny metriky veřejně dostupné, proto se kombinují živá data a fallback informace.
OpenRouter má pro `:free` modely vlastní limity (RPM/RPD). Po dobití kreditu (v aktuální dokumentaci je prahová částka, často kolem 10 USD) se navyšuje denní free limit, ale minutový limit a dostupnost endpointů se může dál lišit model od modelu.
