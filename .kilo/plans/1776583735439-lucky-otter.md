# Plán: Mobile Responsive Upravy - strong_translator.html

## Konkrétní problémy uživatele (popsáno)

1. **Na mobilu na výšku vidí jen levou stranu** - pravý panel není vidět
2. **Po načtení souboru "Zahájit překlad" tlačítko zmizí/není kliknutelné** - přetéká mimo viewport
3. **V aplikaci levý panel (hesla) není vidět** - grid layout 260px se nevejde do ~360px

## Příčiny problémů v kódu

### Problém 1: Setup - .file-row (řádek 94)
```css
.file-row { display: flex; align-items: center; gap: 10px; }  /* bez flex-wrap! */
.start-btn { margin-left: auto; }  /* tlačí doprava, na úzké obrazovce mizí */
```

### Problém 2: Setup - #setup (řádek 40)
```css
#setup { max-width: 680px; margin: 36px auto; padding: 30px; }
```
- 680px se nevejde do ~360px mobilu
- margin 36px uberuje další místo

### Problém 3: App - .split (řádek 279)
```css
.split { display: grid; grid-template-columns: 260px 1fr; }
```
- 260px levý panel + pravý panel se na ~360px nevejde

### Problém 4: App - .log-panel (řádek 237)
```css
.log-panel { height: 50%; }
```
- Zabírá 50% výšky - na mobilu moc

### Problém 5: Header - příliš mnoho prvků
- Statistika: Přeloženo, Zbývá, Celkem, Tokeny, Čas
- Progress bar, input pro G číslo
- 8 tlačítek: Import, AUTO, Dávka, TXT, JSON, Rozsah, Neúspěšné, Reset

## Plán úprav

### Fáze 1: Setup stránka - Opravit tlačítko "Zahájit překlad"

```css
/* Přidat do CSS */
@media (max-width: 600px) {
  #setup { max-width: 100%; margin: 10px auto; padding: 16px; }
  
  .file-row { 
    display: flex; 
    flex-wrap: wrap;  /* NOVÉ - povolit zalamování */
    gap: 8px; 
  }
  
  .file-row > * { flex: 1 1 auto; min-width: 120px; }
  
  .start-btn { 
    width: 100% !important;  /* NOVÉ - button na celou šířku */
    margin-left: 0 !important;
    min-width: unset;
  }
}
```

### Fáze 2: App Layout - Změnit split na flex column

```css
@media (max-width: 600px) {
  .split { 
    display: flex; 
    flex-direction: column;  /* NOVÉ - vertikální layout */
  }
  
  .list-pane { 
    width: 100%; 
    height: 45%;  /* Levý panel nahoře */
    border-right: none; 
    border-bottom: 1px solid var(--brd); 
  }
  
  .detail-pane { 
    width: 100%; 
    height: 55%;  /* Pravý panel dole */
  }
  
  .log-panel { 
    height: 0; 
    display: none;  /* NOVÉ - skrýt log panel na mobilu */
  }
  .log-panel.show-mobile { 
    height: 30%; 
    display: block;  /* Pro přepínání */
  }
}
```

### Fáze 3: Header - Zjednodušit

```css
@media (max-width: 600px) {
  .hdr { padding: 8px 10px; gap: 6px; }
  
  .stat:not(:first-child) { display: none; }  /* Skrýt všechnystat kromě prvního */
  .stat:first-child { display: block; }
  
  .hdr-title { font-size: 10px; }
  
  /* Jeden hlavníAkce */
  .pbar-wrap { width: 100%; order: 10; margin-top: 4px; }
  
  /* Tlačítka pod menu */
  .hdr-btns { 
    width: 100%; 
    justify-content: center; 
    gap: 4px;
  }
  
  .hbtn { 
    font-size: 9px; 
    padding: 4px 6px; 
  }
}
```

Nebo alternativně - hamburger menu pro sekundární tlačítka.

### Fáze 4: List Item - Větší dotykové cíle

```css
@media (max-width: 600px) {
  .list-item { height: 44px; }  /* Větší než 32px */
  .li-check { width: 20px; height: 20px; }  /* Větší checkbox */
  .li-key { font-size: 10px; }
  .li-greek { font-size: 11px; }
}
```

### Fáze 5: Detail pane - Lepší zobrazení

```css
@media (max-width: 600px) {
  .detail-scroll { padding: 12px 14px; }
  .entry-greek { font-size: 16px; }
}
```

## Shrnutí změn

| Místo | Problém | Řešení |
|-------|---------|---------|
| Setup .file-row | Tlačítko mizí | `flex-wrap: wrap` + `width: 100%` pro .start-btn |
| Setup #setup | Přetéká | `max-width: 100%` + menší padding |
| App .split | 260px se nevejde | `flex-direction: column` na mobilu |
| App .log-panel | 50% výšky | Skrýt na mobilu |
| Header | Přetéká | Zjednodušit, skrýt statistiky |
| List item | 32px malé | 44px pro dotyk |
