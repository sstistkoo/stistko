# Plan: Opravy pro mobile resizable panely v strong_translator.html

## Krok 1: Zobrazit resize handle i při zavřeném levém panelu

**Problem:** Na řádku 1167 je `resizeHandle.style.display = 'none'` - to skrývá handle.

**Oprava:**Odstranit nebo zakomentovat řádek 1167.

```
--- strong_translator.html, řádky 1162-1173 (else větev):
  } else {
    pane.classList.add('hidden');
    btn.textContent = '▶';
    if (isMobile) {
      pane.style.height = '0';
      // resizeHandle.style.display = 'none';  // ODSTRANIT
      logPanel.style.height = '100%';
      logPanel.style.flex = '1';
      detailPane.style.height = 'auto';
      detailPane.style.flex = '1';
    }
  }
```

Po této opravě:
- Handle bude stále viditelný (display: flex z CSS)
- Přesouvá se nahoru mezi list-pane a detail-pane
