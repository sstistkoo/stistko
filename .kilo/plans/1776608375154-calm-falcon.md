# Plan: Přidání OpenRouter providera s dynamickým načítáním free modelů

## Cíl
Přidat do strong_translator.html možnost použít OpenRouter s automatickým načtením aktuální nabídky free modelů.

## Kroky

### 1. Přidat funkci pro načtení seznamu free modelů z OpenRouter API
- Nová async funkce `loadOpenRouterFreeModels()`
- Volá endpoint `https://openrouter.ai/api/v1/models?free=true` (nebo filtruje podle `:free`)
- Parsuje odpověď a vrací pole dostupných modelů

### 2. Upravit UI pro dynamické načítání modelů
- V `onProviderChange()` - při výběru OpenRouter zavolat načtení modelů
- Přidat placeholder "Načítám..." během načítání
- Naplnit select element se seznamem modelů

### 3. Upravit API volání pro OpenRouter
- Ověřit, že existující `callAIWithRetry`/call funkce správně volá OpenRouter endpoint
- Případně přidat specifické hlavičky pro OpenRouter pokud chybí

### 4. Aktualizovat popis providera v UI
- Změnit label na "OpenRouter (openrouter.ai) - zdarma"
- Upravit placeholder pro API key

## Poznámky
- OpenRouter free modely mají v ID příponu `:free` (např. `meta-llama/llama-3.1-8b-instruct:free`)
- Nejpopulárnější free modely (2026): NVIDIA Nemotron 3 Super, Gemma 4 31B, MiniMax M2.5, Qwen3, DeepSeek R1
- API endpoint: `https://openrouter.ai/api/v1/chat/completions`
