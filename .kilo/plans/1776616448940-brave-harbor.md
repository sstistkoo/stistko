# Plán: Přidání angličtiny do Strong Translator

## Aktuální stav
Uživatel má:
- **strong_greek_detailed.txt** - řecké heslá (G1, G2...) ✓
- **strong_hebrew_detailed.txt** - hebrejské heslá (H1, H2...) ✓
- **strongsgreek.json** - JSON s řeckými daty (unicode, definition, kjv_def, derivation)
- **stronghebrew.json** - JSON s hebrejskými daty (hebrew, xlit, definitions)

Chybí:
- **strong_english_detailed.txt** - anglické definice ✗

## Cíl
Vytvořit TXT soubor s anglickými definicemi ve stejném formátu jako existující TXT soubory.

## Formát TXT souboru (existující)
### Řecký (strong_greek_detailed.txt):
```
G1 | Α
Definition: the first letter of the alphabet; figuratively, only (from its use as a numeral) the first:
KJV Meaning: Alpha.
Origin: of Hebrew origin;
```

### Hebrejský (strong_hebrew_detailed.txt):
```
H1 | אב
Definition: chief, (fore-) father(-less), [idiom] patrimony, principal...
Meaning: 1) father of an individual; 2) of God as father of his people...
```

## Řešení

### Možnost 1: Jeden anglický soubor (doporučeno)
Vytvořit `strong_english_detailed.txt` který obsahuje anglické definice z obou zdrojů (řeckého i hebrejského) - protože Strong's slovník je univerzální a definice jsou stejné bez ohledu na jazyk.

### Možnost 2: Dva anglické soubory
- `strong_english_greek.txt` - z strongsgreek.json
- `strong_english_hebrew.txt` - z stronghebrew.json

## Realizace
Vytvořit Python skript `generate_english_txt.py` který:
1. Načte strongsgreek.json → extrakce definition + kjv_def + derivation
2. Načte stronghebrew.json → extrakce definitions
3. Vytvoří TXT ve formátu:
   ```
   E1 | (Greek: Α / Hebrew: אב)
   Definition: the first letter...
   Meaning: Alpha.
   Origin: of Hebrew origin;
   ```
   
Nebo jednodušeji - použít anglické definice z řeckého slovníku jako "master" protože Strong# je stejný.

## Aktuální zadání
Vytvořit **strong_english_detailed.txt** který obsahuje anglické definice z obou zdrojů, oddělené podle čísel:
- **G1, G2, G3...** = řecká slova (z strongsgreek.json)
- **H1, H2, H3...** = hebrejská slova (z stronghebrew.json)

### Formát výstupu
```
G1 | Α (Greek) / אב (Hebrew)
Definition: the first letter of the alphabet; figuratively, only (from its use as a numeral) the first:
Meaning: Alpha.
Origin: of Hebrew origin;

H1 | אב
Definition: chief, (fore-) father(-less), [idiom] patrimony, principal...
Meaning: 1) father of an individual; 2) of God as father of his people...
```

### Kroky realizace
1. Vytvořit Python skript `generate_english_txt.py`
2. Načíst strongsgreek.json → pro každý entry:
   - ID → G{number}
   - greek.unicode → řecké slovo
   - definition → Definice
   - kjv_def → Meaning
   - derivation → Origin
3. Načíst stronghebrew.json → pro každý entry:
   - ID → H{number}
   - hebrew → hebrejské slovo
   - definitions → Meaning (joined)
   - Přidat reference na řecké ekvivalenty pokud existují
4. Uložit do strong_english_detailed.txt
5. Formát: "G{num} | {greek_word}" a "H{num} | {hebrew_word}"
